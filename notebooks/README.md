# Jupyter Notebooks

This directory contains Jupyter notebooks for data exploration, model development, and analysis.

## Structure

- `data_exploration/` - Exploratory data analysis
- `model_training/` - Model training and evaluation
- `evaluation/` - Model performance analysis
- `visualization/` - Data visualization examples

## Usage

```bash
# Install Jupyter
pip install jupyter notebook

# Start Jupyter
jupyter notebook
```

## Notebooks

### Data Exploration
- Climate data analysis
- Data quality checks
- Feature engineering experiments

### Model Training
- Model development
- Hyperparameter tuning
- Model comparison

### Evaluation
- Performance metrics
- Error analysis
- Validation studies
